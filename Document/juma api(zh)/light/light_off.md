
#关闭调光灯
***
##函数声明
```
void light_off(void);
```

***
##函数功能
在调光灯已经被打开的情况下可以调用本函数来关闭调光灯


***
##函数参数
参数    | 数据类型   | 说明
:----- | :-------- | :------
*返回值*  | 无    | 无

***
##函数举例

```	
	//当前函数
	void foo()
	{
		...
		
		//配置调光灯 其驱动引脚为21,22,23 其驱动电平为低
		light_setup((uint8_t *)&light_init_config, 0);
	
		...
	
		//当配置好调光灯的引脚和驱动电平后，则可以直接调用本函数打开调光灯
		light_on();
		
		...
		
		//在调光灯已经打开的情况下，调用本函数可以关闭调光灯。
		light_off();
		
		...
	}
```
